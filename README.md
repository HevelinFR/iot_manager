## Trabalho de Conclusão de Curso de graduação

A Internet das Coisas (IoT) é um conceito tecnológico que possibilita a conexão de dispositivos físicos à internet, permitindo a troca de dados e a comunicação entre eles. Com o avanço dessa tecnologia, surgem novas oportunidades e desafios, especialmente em cenários críticos, nos quais a confiabilidade e o correto funcionamento dos dispositivos são essenciais.

Este trabalho insere-se no contexto da Internet das Coisas aplicada a ambientes críticos, definidos como aqueles em que sensores IoT devem apresentar comportamentos previsíveis, mantendo suas métricas dentro de limites predefinidos. Propõe-se o desenvolvimento de uma aplicação web para o cadastro e gerenciamento de ambientes e dispositivos IoT, bem como para a definição, detecção e envio de alertas em situações críticas. O trabalho descreve o processo de desenvolvimento e apresenta uma versão funcional da aplicação, demonstrando a viabilidade da solução proposta e apontando possibilidades para trabalhos futuros.
